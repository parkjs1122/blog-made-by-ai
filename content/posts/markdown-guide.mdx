# Markdown Syntax Guide

This post demonstrates various Markdown syntax elements that you can use in your blog posts.

## Headings

# H1 Heading
## H2 Heading
### H3 Heading
#### H4 Heading

## Text Formatting

You can make text **bold**, *italic*, or ***bold and italic***.

You can also use ~~strikethrough~~.

## Lists

### Unordered List

- Item 1
- Item 2
  - Nested item 2.1
  - Nested item 2.2
- Item 3

### Ordered List

1. First item
2. Second item
3. Third item

## Links and Images

[This is a link](https://example.com)

## Code

Inline code: `const greeting = "Hello, World!"`

Code block:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`)
}

greet("World")
```
